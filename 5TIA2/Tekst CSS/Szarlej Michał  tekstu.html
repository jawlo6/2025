.kolor1 { color: red; } .kolor2 { color: rgb(0, 150, 0); } .kolor3 { color: #3366cc; } 2. Wyrównywanie tekstu (text-align)

Tekst wyrównany do lewej.

Tekst wyśrodkowany.

Tekst wyrównany do prawej.

Tekst justowany (równy po obu stronach).

.wyrownanie1 { text-align: left; } .wyrownanie2 { text-align: center; } .wyrownanie3 { text-align: right; } .wyrownanie4 { text-align: justify; } 3. Wyrównanie w pionie (vertical-align)

Obrazek i tekst obok.

Tekst podkreślony
