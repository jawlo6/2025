Przykłady Formatowania Tekstu w CSS

1. Kolor - Właściwość color

Zmieniamy kolor tekstu używając różnych formatów kodowania kolorów.

Kolor po nazwie (red).

Kolor po kodzie HEX (#34A853).

Kolor po kodzie RGB (niebieski).

Kolor po kodzie RGBA (przezroczysty róż).

2. Wyrównywanie - Właściwość text-align

Kontroluje położenie tekstu wewnątrz bloku (lewo, środek, prawo, justowanie).

1. Wyrównanie do lewej (default).

2. Wyrównanie do środka.

3. Wyrównanie do prawej.

4. Justowanie. Ten długi tekst będzie rozciągnięty, aby linijki kończyły się równo po obu stronach. Jest to często używane w treściach, które mają przypominać książki lub artykuły.

3. Wyrównanie w Pionie - Właściwość vertical-align

Używane do ustawiania małych elementów (np. ikon, indeksów) w pionie względem linii bazowej tekstu.

1. Wyrównanie ikony do środka.

2. Indeks górny: 102

3. Indeks dolny: H2O

4. Dekoratory - Właściwość text-decoration

Dodaje lub usuwa linie wokół tekstu (podkreślenie, przekreślenie, itp.).

1. Link bez podkreślenia (text-decoration: none).

2. Podkreślenie (underline).

3. Przekreślenie (line-through).

4. Własna dekoracja (underline wavy red 2px).

5. Transformacje - Właściwość text-transform

Automatycznie zmienia wielkość liter w tekście, niezależnie od tego, jak zostały wpisane w HTML.

1. ten tekst będzie dużymi literami.

2. TEN TEKST BĘDZIE MAŁYMI LITERAMI.

3. każde słowo z dużej litery.

6. Odstępy - Spacing (letter-spacing, line-height, itp.)

Kontrola nad odstępami między literami, słowami i liniami tekstu.

1. Odstępy między Literami (letter-spacing).

2. Odstępy między słowami (word-spacing).

3. Duża Interlinia (line-height: 2.0). Akapit jest bardziej przewiewny i łatwiejszy do czytania dzięki większej przestrzeni między liniami tekstu.

4. Wcięcie pierwszej linii (text-indent: 50px). Tylko pierwszy wiersz ma wcięcie, reszta jest normalnie.

7. Cienie - Właściwość text-shadow

Dodaje efekt cienia do tekstu (X, Y, rozmycie, kolor).

1. Prosty, rozmyty cień

2. Twardy cień (retro)

3. Tekst z obrysem

4. Efekt NEON
